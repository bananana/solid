Alerts

Alerts are usually rendered using Flask's flash() method, but they can always be rendered by applying flash class to a <div>.

Here are a few examples. Adding success or good class generates a success alert. Adding warning or unknown class generates a warning alert. Adding danger, bad or error class generates an error alert. By default all alerts have their width set to 100%, so they take up an entire parent block. Don't forget to add role="alert" property for accessability.

Additionaly you can add a close button to an alert by including <a class="flash-close"><span class="oi" data-glyph="x"></span></a> inside the flash container.
