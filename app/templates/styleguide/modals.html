Modals

Modals are dialog boxes that are displayed on top of normal content. Click on the button below to view a demo modal.

{{ render_modal('modal-demo', 'Modal Demo', 'Modal demo content', 'Click here to see a modal', 'button') }}

Note: using modals breaks accessability. That is: screen readers will not interpret them correctly.

You can either render modals manually or use the included macro. Below is an example manual implementation. First, import modal.js file inside {% block scripts %}.

<script src="{{ url_for('static', filename='js/modal.js') }}"></script>

Then, somewhere inside {% block scripts %}, create a button like so:

<a class="button button-modal" data-target="modal-demo">Modal Demo</a>

The modal itself is always hidden by default and you can put it anywhere inside <body>. Note that trigger button's data-target parameter and modal's id match up. This is so the script knows which button triggers which modal.

<div id="modal-demo" class="modal">
    <div>
        <h3 class="u-pull-left">Modal Demo</h3>
        <a href="#" class="button-modal-close u-pull-right">
            <img src="/static/open-iconic/svg/x.svg" alt="X" />
        </a>
        <p class="u-cf">Modal demo content</p>
    </div>
</div>
