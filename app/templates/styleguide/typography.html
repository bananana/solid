Typography

Just as vanilla Skeleton, we're using Raleway font, which is served locally. Type is set using rems, so all font sizes are based on a single fots-size property defined for <html> element. Generally, that definition should not be changed. This is done to keep font-size relationships consistent. All measurements are base 10, so <h1> with 5.0rem means 50px.

The base font size is 15rem (15px) with 1.6rem line height (24px). Here are some examples of basic tags:

Paragraphs

Simply use the <p> tag. Here's an example:

Normal size
Est et ad rem unde a commodi. Nemo velit deserunt molestias velit. Dolor nesciunt recusandae magni provident eos. Dolore ea veritatis quod alias dolorum facere animi sunt. Ea voluptatem voluptatem perspiciatis quia voluptatem veritatis ut. Perferendis optio modi quas sunt natus.

Small size
A et tenetur at non quo vitae eveniet. Tempora quos aut placeat nam ullam est qui repellat. Praesentium consequatur alias aut doloribus harum rerum facere aut. Perspiciatis saepe aut nulla. Expedita minima qui sunt facilis voluptas possimus. Dolor officiis natus sint perferendis.

<-- A standard, normal size paragraph -->
<p><strong>Normal size</strong><br>
Est et ad rem unde a commodi. [...]</p>
        
<-- A small paragraph with a more subdued color -->
<p><small><strong>Small size</strong><br>
A et tenetur at non quo vitae eveniet. [...]</p>

Lists

Unoredered lists have a circle decoration by default. Nesting for more that 2 levels is not currently supported.

Unordered list:

  • Item 1
  • Item 2
    • Subitem 1
    • Subitem 2
  • Item 3

Ordered list:

  1. Item 1
  2. Item 2
    1. Subitem 1
    2. Subitem 2
  3. Item 3

Code

Code styling is basic, without any bells and whistles. For inline code wrap text in the <code> tag. For block code, use the <pre> tag, which will preserve indentation and white space. The < and > characters have to be represented using ASCII codes &lt; and &gt; respectively.

<pre>
.example-class {
    background-color: #000;
}<pre>
