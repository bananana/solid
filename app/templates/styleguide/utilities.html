Utilities

In addition to Skeleton's helper utilites we've added several others to help with layout creation. Below is a full list of these classes, the details of their implementation can be found in app/static/css/custom.css under "Utilities" section.

Class Explanation
u-pull-right Float right
u-pull-left Float left
u-cf Clear all floats
u-full-width Set element width to 100%, making it take up the entire width of parent element
u-max-full-width Same as u-full-width, only using the max-width property
u-add-margin-top Add a 1em top margin
u-remove-margin-bottom Remove bottom margin by setting that property to 0
u-to-upper Turn text to all caps
u-text-center Align text to center
u-text-right Align text to the right
u-mobile-fixed-bottom Make element stick to the bottom of the screen on mobile devices (width 550px or less)
u-hide-on-mobile Hide element on mobile devices (width 550px or less)
u-show-on-mobile Only display element on mobile devices (width 550px or less)
