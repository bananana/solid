Grid System

The grid system is partially borrowed from Skeleton. It is a 12-column 960px or full-width grid. It is completely responsive.

Full-Width Container

A full-width container is also available. It takes up the whole viewport width and is divided into 12 columns, just like the standard 960px container.

Use the <div class="full-width-container"> instead of the standard Skeleton <div class="container"> to make use of this feature.

<div class="full-width-container">

    <div class="row">
        <div class="three columns">Three</div>
        <div class="four columns">Four</div>
        <div class="five columns">Five</div>
    </div>

</div>

    <div class="two-thirds columns">Two Thirds</div>
</div>
<div class="row">
    <div class="one-half column">One Half</div>
    <div class="one-half column">One Half</div>
</div>

</div>

Column Offsets

Sometimes it is necessary to offset a column by a certain amount. This is pretty easy to do, just use offset-by-[amount] class, where [amount] is the number of columns like; one, two, three and so on all the way to eleven.

Eight columns offset by four
Ten columns offset by two
<div class="container">

<div class="row">
    <div class="eight columns offset-by-four">Eight columns offset by four</div>
</div>
<div class="row">
    <div class="eight columns offset-by-two">Ten columns offset by two</div>
</div>

</div>
