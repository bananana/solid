About

This style guide is here to help new and current developers to keep the UI consistent. If you modify the styles, pease update this guide.

Currently for frontend development we're using:

  • Skeleton CSS boilerplate.
  • Open Iconic icons
  • A few custom social media svg icons
  • Jinja2 templating engine.

Skeleton

Skeleton is not a fremework, it's a boilerplate to create unique style for a website. To keep things easy to maintain it's core is located in:

app/static/Skeleton

All custom additions to it are added to a separate file:

app/static/css/custom.css

Raleway

Currently using the default font that comes with Skeleton, called Raleway. For development all files are server locally from:

app/static/fonts

The font comes in 3 styles; Light, Regular and Semi-Bold (Raleway-Light.ttf, Raleway-Regular.ttf and Raleway-SemiBold.ttf). For detailed specifications visit the Fonts section of this style guide.

Open Iconic

We're using Open Iconic icon set. Throughout the app svg icons are used istead of the icon font. It is located in:

app/static/open-iconic/svg

Additionally, there are a few custom social media icons located in:

app/static/icons

Jinja2

Jinja comes by default with Flask, which is the Python microframework we're using. Detailed documentation can be found here.

Templating conventions used for this project are described in the Templating section.
