Media Queries

Media queries are borrowed directly from Skeleton. All the queries are mobile-first, meaning they use min-width property.

Note: The best way to structure the use of media queries is to create the queries near the relevant code. For example, if you wanted to change the styles for buttons on small devices, paste the mobile query code up in the buttons section and style it there.

Below is a table of all available media queries:

Media Query Explanation
@media (min-width: 400px) {} Larger than mobile
@media (min-width: 550px) {} Larger than phablet
@media (min-width: 750px) {} Larger than tablet
@media (min-width: 1000px) {} Larger than desktop
@media (min-width: 1200px) {} Larger than HD desktop
